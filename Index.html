Evolución de la Web

Web 1.0

La Web 1.0 corresponde a la primera etapa de Internet, caracterizada por páginas estáticas, con poco o ningún nivel de interacción con el usuario.

Su función principal era mostrar información, sin posibilidad de que los visitantes generaran contenido.

Web 2.0

La Web 2.0 marcó un cambio hacia la interactividad y la colaboración. Aquí los usuarios pudieron crear, compartir y comentar contenido.

Se destacan redes sociales, blogs, wikis y plataformas de colaboración.

Web 3.0

La Web 3.0 apunta hacia la descentralización, la inteligencia artificial y el uso de blockchain.

Su objetivo es crear una red más semántica, personalizada y segura.
